# Fictitious Play

Convergence analyzer for classical and smooth (logit) fictitious play in two-player normal-form games.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/fictitious-play.git
cd fictitious-play
make install
make run
make test
```

See [docs/](../docs/fictitious-play.tex) for the math.
